# customer_behaviour_analysis
This project focuses on analyzing customer behavior under different circumstances to uncover actionable insights for business decision-making. Using sales and interaction data, I explored how customers respond to various factors such as pricing, promotions, product categories, and seasonal trends.

🛒 Customer Shopping Behavior Analysis
📌 Overview
This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, product preferences, subscription behavior, and customer segments. The goal is to support data‑driven business decisions through structured analysis and interactive dashboards.

📂 Dataset
Size: 3,900 rows × 18 columns

Features:

Customer demographics (Age, Gender, Location, Subscription Status)

Purchase details (Item, Category, Amount, Season, Size, Color)

Shopping behavior (Discounts, Promo Codes, Frequency, Reviews, Shipping Type)

Missing Data: 37 values in Review Rating column

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, EDA

PostgreSQL / MySQL / SQL Server – Business transaction queries

Power BI – Interactive dashboards

Gamma – Automated presentation slides

MS Excel – Quick checks and preprocessing
